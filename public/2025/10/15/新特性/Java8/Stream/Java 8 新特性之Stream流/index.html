Java 8 新特性之Stream流

是什么是Stream流

java.util.stream.Stream

Stream流和传统的IO流，它们都叫流，却是两个完全不一样的概念和东西。

流可以简单的说是处理数据集合的东西，可以申明式流式API来处理集合，而不是写一个逻辑实现。

流分类

流分为顺序流及并行流，顺序流即每个指令按顺序执行，并行流即集合里面的操作并行执行。
