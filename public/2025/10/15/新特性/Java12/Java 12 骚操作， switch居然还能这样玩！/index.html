Java 12 switch 有以下几点特色：

  • 箭头语法 ->，类似 Java 8 中的 Lambda 表达式；
  • 可以直接返回值给一个变量，并且可以不用 break 关键字；
  • case 条件，多个可以写在一行，用逗号分开；
  • 可以省略 break 关键字；
